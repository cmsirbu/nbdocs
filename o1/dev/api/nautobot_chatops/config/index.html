nautobot_chatops.config.load(config_file_name='pyproject.toml', config_data=None)

Loads passed dynamic settings data using config data or from a TOML configuration file (pyproject.toml).

When loaded from file, the settings for this app are expected to be in [tool.nautobot-chatops] in TOML if nothing is found in the config file or if the config file do not exist, the default values will be used.

Args:

config_file_name (str, optional): Name of the configuration file to load. Defaults to “pyproject.toml”. config_data (dict, optional): dict to load as the config file instead of reading the file. Defaults to None.

nautobot_chatops.config.load_or_exit(config_file_name='pyproject.toml', config_data=None)

Loads passed dynamic settings data using config data or from a TOML configuration file (pyproject.toml).

If a validation error is found, it will print out the respective error and exit from the app.

When loaded from file, the settings for this app are expected to be in [tool.nautobot_chatops] in TOML if nothing is found in the config file or if the config file do not exist, the default values will be used.

Args:

config_file_name (str, optional): Name of the configuration file to load. Defaults to “pyproject.toml”. config_data (dict, optional): dict to load as the config file instead of reading the file. Defaults to None.
