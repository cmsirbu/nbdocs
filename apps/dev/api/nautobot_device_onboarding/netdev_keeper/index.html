Get the interface name and prefix length for the management interface.

Locate the interface assigned with the hostname value and retain the interface name and IP prefix-length so that we can use it when creating the IPAM IP-Address instance.

Note that in some cases (e.g., NAT) the hostname may differ than the interface addresses present on the device. We need to handle this.
